# 🌦️ SkyCast Weather App

SkyCast is a modern weather application built with **React.js** and **Vite** that provides real-time weather information for any city. It uses a Weather API to display accurate weather details with a clean and responsive user interface.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- 💧 Humidity details
- 🌬️ Wind speed
- ☁️ Weather condition with icons
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6)
- HTML5
- CSS3
- Weather API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Dasarihemavathi/Dasarihemavathi-skycast_weather_app.git
```

### Navigate to the project

```bash
cd Dasarihemavathi-skycast_weather_app
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
